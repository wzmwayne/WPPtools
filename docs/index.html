WPStools v1.0.0

在触屏设备上使用 WPS 演示放映幻灯片时，提供悬浮翻页控件和工具栏，无需键盘即可操作。

功能

  • 屏幕两侧半透明翻页按钮（上一页 / 下一页）
  • 底部工具栏：切换鼠标 / 笔 / 橡皮擦模式
  • 一键退出放映，支持多级重试（Esc → Alt+D → 强杀）
  • 配置面板：调节透明度、尺寸、边缘距离等
  • 通过 Task Scheduler 实现开机自启（非注册表）
  • 自动检测 WPS 放映，显示/隐藏控件

下载

WPPtools_offline.exe 离线安装包（含 Qt 运行时，~50 MB） WPPtools_online.exe 在线安装包（自动下载 Qt 运行时，~5 MB）

使用

  1. 运行 WPPTouchHelper.exe（后台常驻，无托盘图标）
  2. 打开 WPS 演示并开始放映幻灯片
  3. 屏幕两侧出现翻页按钮，底部出现工具栏
  4. 运行 WPStoolsPanel.exe 打开配置面板

构建

需 MSVC 2022 BuildTools + Qt 6.x（msvc2022_64）。运行 build.bat 一键构建。
